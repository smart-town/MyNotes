# Groovy 认识

Groovy 是一种基于 JVM 的敏捷开发语言，它结合了 Python、Ruby、Samlltalk 的许多强大特性，Groovy 代码能够很好地与 Java 代码结合，也能用于扩展现有代码。由于其运行在 JVM 上的特性，Groovy 可以使用其他 Java 语言编写的库。

## 简介

Groovy 是用于 Java 虚拟机的一种敏捷的动态语言，它是一种成熟的面向对象编程语言，既可以用于面向对象编程，又可以用作纯粹的脚本语言，使用该语言不必编写过多的代码，同时又具有闭包和动态语言中的其他特性。

它是 JVM 的一个替代语言，即可以用 Groovy 在 Java 平台上进行 Java 编程，使用方式基本与 Java 代码方式相同。该语言特别适合与 Spring 的动态语言支持一起使用，设计时充分考虑了 Java 集成，这使得 Groovy 与 Java 的互操作很容易。
