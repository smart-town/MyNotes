# 常见环境变量:eye:

## ANDROID 相关

 变量名     |    默认值  |    含义
 --------- | --------- | ------------
 `ANDROID_SDK_HOME` | `%USER_PROFILE%\.android` | 创建本地模拟器时使用
 `ANDROID_SDK_ROOT` | - | 设置 Android SDK 目录
 `ANDROID_HOME` | - | 设置 Android SDK 目录（已废弃）

### GRADLE

 变量名     |    默认值  |    含义
 --------- | --------- | ------------
`GRADLE_USER_HOME` | `%USER_PROFILE%\.gradle` | gradle 缓存目录
