# 宏

## 编译预处理指令

- `#` 开头的是编译预处理指令
- 它们并不是 C 语言的成分，但是 C 语言程序离不开它们
- `#define` 用来定义一个宏

### 预定义的宏

- `__LINE__` 源代码行号
- `__FILE__` 文件名
- `__DATE__` 编译日期
- `__TIME__`
- `__STDC__`

### 带参数的宏

如`#define cube(x) ((x)*(x)*(x))`

- 大型程序中使用非常普遍
- 可以非常复杂，如产生函数
    - 在`#`和`##`的帮助下
- 存在中西方差异
- 部分宏会被 inline 函数替代
