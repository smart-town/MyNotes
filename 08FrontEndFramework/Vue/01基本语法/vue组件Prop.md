# Prop

## Prop 大小写

HTML 中特性名是不区分大小写，意味着浏览器会将所有大写字符转换为小写字符。因此当你使用 DOM 中的模板时，camelCase 驼峰命名法的 prop 需要使用其等价的 kebab-case 短横线分隔命名。

注意，如果使用的是**字符串模板**，那么这个限制就不存在了
